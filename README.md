# YouTube Video Manager

A simple Node.js CLI tool to manage YouTube videos: download thumbnails, store video info in a JSON database, add tags, remove videos, and view your database.

## Features

- Add YouTube videos by URL or ID
- Automatically download video thumbnails
- Store video data (title, creator, duration, link, thumbnail, tags) in `data.json`
- Remove videos with automatic ID reindexing
- View database in a clean, readable format
- Add optional tags for each video
- Single-run CLI menu
